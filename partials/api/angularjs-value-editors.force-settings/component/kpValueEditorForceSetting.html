kpValueEditorForceSetting

  1. - component in module angularjs-value-editors.force-settings

This component is used for override options and validations for specified type of value editor. It requires kpValueEditorForceSettings

Dependencies

  • kpValueEditorForceSettings

Usage

<kp-value-editor-force-setting
    [type="CustomValueEditorType"]
    [options="OPTIONS"]
    [validations="VALIDATIONS"]>
</kp-value-editor-force-setting>

Arguments

Param Type Details
type
(optional)
 CustomValueEditorType

Type of value editor. If type is not specified, options will be passed to all editors.
options
(optional)
 OPTIONS

Options
validations
(optional)
 VALIDATIONS

Validations (not implemented yet)
