kpModelEquals

  1. - directive in module angularjs-value-editor.password

Directive add equals validator. If input value is not equal to model value, it will fails.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY data-kp-model-equals
	data-kp-model-equals="string">
	...
</ANY>

Arguments

Param Type Details
kpModelEquals String

Comparing value.
