sliderModelTransformer

  1. - directive in module angularjs-value-editor.range

Directive transforms ngModel from object {from: XXX, to: YYY} to array [XXX, YYY] If selected values are minimum and maximum values from slider and sliderModelTransformer param is true, then model will be transformed to null.

Dependencies

  • ngModel
  • tdSlider

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <td-slider data-slider-model-transformer
	data-slider-model-transformer>
	...
</td-slider>

Arguments

Param Type Details
sliderModelTransformer Boolean

If true, transform selected extremes as null.
