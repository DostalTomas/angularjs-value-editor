kpIndeterminateCheckbox

  1. - directive in module angularjs-value-editor

Directive set indeterminate state based on input.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    	<kp-indeterminate-checkbox>
	...
</kp-indeterminate-checkbox>
  • as attribute: 
    <ANY data-kp-indeterminate-checkbox
	data-kp-indeterminate-checkbox>
	...
</ANY>

Arguments

Param Type Details
kpIndeterminateCheckbox Boolean

True if set to indeterminate state.
