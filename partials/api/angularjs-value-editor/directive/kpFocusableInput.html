kpFocusableInput

  1. - directive in module angularjs-value-editor

Directive which provides API (FocusableInputAPI) to focus Input Element with this directive.

API

Method name Description
focusInput(): void Triggers focus on the element

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY data-kp-focusable-input
	data-kp-focusable-input="expression">
	...
</ANY>

Arguments

Param Type Details
kpFocusableInput expression

Expression evaluated in link phase. Publishing $api as parameter with API reference.
