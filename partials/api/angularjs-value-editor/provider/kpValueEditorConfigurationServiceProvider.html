Provider for kpValueEditorConfigurationService

Properties

  • debugMode

    Boolean

    Enable / disable debug mode. It show / hide information section below value editor.

  • preciseWatchForOptionsChanges

    Boolean

    It enables deep watching for changes in value editors options. If watching for changes is not needed, it's recommended set it to false due to high system requirements. (It makes deep equal of options in each digest cycle).
