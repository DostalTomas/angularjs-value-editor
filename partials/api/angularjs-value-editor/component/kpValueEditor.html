kpValueEditor

  1. - component in module angularjs-value-editor

Generic value editor depends on type:

  • text: textValueEditor
  • number: numberValueEditor

Dependencies

  • ngModel.NgModelController

Usage

<kp-value-editor
    editor-id="string"
    editor-name="string"
    placeholder="string"
    type="string"
    disabled="boolean"
    visible="boolean"
    validations="ValueEditorValidations"
    options="ValueEditorOptions">
</kp-value-editor>
