htmlValueEditor

  1. - component in module angularjs-value-editor

Value editor for formatted text input.

Component uses Trumbowyg editor, because it is lightweight, (almost) libraries agnostic with possibility to be, in near future, jQuery less.

It is required to attach also table and colors plugins - more info how to do it can be find in Trumbowyg site. For attaching SVGs via webpack use this configuration:

$.trumbowyg.svgPath = require('trumbowyg/dist/ui/icons.svg');

Supported options: HtmlValueEditorOptions

Supported validations: ValueEditorValidations

Dependencies

  • ngModel.NgModelController
  • kpValueEditor
  • jQuery
  • Trumbowyg dependencies and plugins.

Usage

<html-value-editor>
</html-value-editor>
