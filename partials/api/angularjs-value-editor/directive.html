Directive components in angularjs-value-editor

Name Description
disableNgAnimate

Fixes bug in ui-select consisting of setting opacity: 0 and not setting opacity: 1 if ngAnimate is present and disabled by classNameFilter.
kpFocusableInput

Directive which provides API (FocusableInputAPI) to focus Input Element with this directive.
kpIndeterminateCheckbox

Directive set indeterminate state based on input.
modelToJsonViewFormatter

Directive formatting model value to JSON.
patternDescriptionTooltip

This directive adds tooltip on input, if valueEditor has defined non-empty localization named patternDescription.
