TextValueEditorOptions

  1. - type in module angularjs-value-editor.text

Extends ValueEditorOptions

Defaults: textValueEditorDefaultOptions

Properties

  • type

    TTextValueEditorType

    Input type. Possible values are text, textarea, rich-textarea, email, url, tel.

  • aceOptions

    Object

    Options for ACE editor. Applicable only if type is 'rich-textarea'.

  • prefix

    String

    Non-editable prefix before input element.

  • suffix

    String

    Non-editable prefix after input element.

  • includePrefixAndSuffixToModel

    Boolean

    If true, prefix and suffix will be appended to the model.

  • trim

    Boolean

    If true, model will be trimmed.

  • switchToTextareaThreshold

    Boolean

    If type is text and length of input text is greater or equal than this threshold, input will force to textarea type. If 0, input never switch to textarea type.
