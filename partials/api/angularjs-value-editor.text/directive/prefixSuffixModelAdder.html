prefixSuffixModelAdder

  1. - directive in module angularjs-value-editor.text

This directive adds prefix and suffix to model if they are set together with includePrefixAndSuffixToModel.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    	<prefix-suffix-model-adder>
	...
</prefix-suffix-model-adder>
  • as attribute: 
    <ANY data-prefix-suffix-model-adder>
	...
</ANY>
