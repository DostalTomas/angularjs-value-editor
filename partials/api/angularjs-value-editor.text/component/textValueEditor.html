textValueEditor

  1. - component in module angularjs-value-editor.text

Value editor for text input. Depending on type are four versions:

  • text

    Common text input.

  • textarea

    Textarea value editor.

  • rich-textarea.

    ACE editor.

  • email

    Email input. If pattern validation is not given, default is used.

  • tel

    Telephone number input.

  • url

    URL input.

Supported options: TextValueEditorOptions

Supported validations: TextValueEditorValidations

Dependencies

  • ngModel.NgModelController
  • kpValueEditor

Usage

<text-value-editor>
</text-value-editor>

Example

<main>
     <kp-value-editor type="'text'" ng-model="model"></kp-value-editor>
     <div>{{model}}</div>
</main>
