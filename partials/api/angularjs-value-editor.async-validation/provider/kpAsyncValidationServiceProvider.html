  1. - kpAsyncValidationService

Provider for async validations.

Methods

  • setValidationFunction(validationFunction);

    Setter for validation function. Validation function is called every time when async validation is triggered. Function returns Promise depending on validation result. If validation failed, promise will be rejected. Rejection reason should be string which will be displayed as validation error message.

    If validation succeeds, promise will be resolved.

    Validation function is called using Injector.invoke() function, so it accepts some injectable parameters.

    Injectable argument name       Description
    $model Current model
    $formModel If sendWholeForm option is true, it contains form model
    $additionalParameters Current model

    Parameters

    Param Type Details
    validationFunction ValidationFunction

    User-specified validation function.
