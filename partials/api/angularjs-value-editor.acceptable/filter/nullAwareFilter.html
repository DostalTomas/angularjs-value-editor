nullAwareFilter

  1. - filter in module angularjs-value-editor.acceptable

It is almost classic angular filter filter, but if search is null it returns original data.

Usage

In HTML Template Binding

{{ nullAwareFilter_expression | nullAwareFilter : search}}

In JavaScript

$filter('nullAwareFilter')(items, search)

Arguments

Param Type Details
items Array.<T>

Array of items
search String

Filter expression
