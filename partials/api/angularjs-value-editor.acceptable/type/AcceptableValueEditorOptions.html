AcceptableValueEditorOptions

  1. - type in module angularjs-value-editor.acceptable

Extends ValueEditorOptions

Default value: acceptableValueEditorDefaultOptions

Properties

  • acceptableValues

    Array.<VALUE>

    Array of predefined values.

  • multiselectable

    Boolean

    If true, value editor will accept multiple values and init model as empty array if not.

  • optionsTemplate

    String

    Angular template for displaying options. Current option is accessible via $item variable name.

  • singleSelectedValueTemplate

    String

    Angular template for displaying selected value in single select mode. Current option is accessible via $select.selected variable name.

  • multiSelectedValueTemplate

    String

    Angular template for displaying selected value in multiple select mode. Current option is accessible via $item variable name.

  • equalityComparator

    VALUE): boolean function(VALUE, VALUE): boolean

    Custom equality comparator.

  • searchable

    Boolean

    If true, select component will have search input. Applicable only for select mode.

  • reorderable

    Boolean

    If true, multi-select component will have capability for manual ordering selected items. Applicable only for multiple select mode.

  • sortComparator

    VALUE): number function(VALUE, VALUE): number

    If defined, options will be sorted using this comparator function.

  • sortModel

    Boolean

    It true, model will be sorted using comparator. Applicable only for multiselectable mode.

  • switchToCheckboxesThreshold

    Number

    If count of options is bigger then this threshold, value editor switches into checkbox mode. If threshold is 0, value editor forces into checkbox mode. Applicable only for multiselectable, non-reorderable mode.

  • showFirstCount

    Number

    If count of options is bigger than this value, value editor shows only given count checkboxes and rest of options is hidden. Applicable only for multiselectable, checkbox mode.

  • selectedFirst

    Boolean

    If true, selected options will be moved on top of options. Applicable only for multiselectable, checkbox mode.

  • modelAsArray

    Boolean

    If true, model will be array also if multiple = false. In this case array will contain only one element.
